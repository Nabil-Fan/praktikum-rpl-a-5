# Backlog EcoEats

|--------|------------------------------------------------------------------------------------------------------------------------------------|------------------|---------------|
| US-01  | Sebagai pengguna baru, saya ingin mendaftarkan akun dengan email dan password, agar saya dapat mengakses platform EcoEats.        | User             | Must-have     |
| US-02  | Sebagai pengguna terdaftar, saya ingin login ke akun saya, agar saya dapat memesan makanan surplus.                                | User             | Must-have     |
| US-03  | Sebagai mitra merchant, saya ingin login ke akun merchant saya, agar saya dapat mengelola menu surplus toko saya.                  | Mitra Merchant   | Must-have     |
| US-04  | Sebagai mitra merchant, saya ingin menambahkan menu surplus beserta harga diskon dan stok tersedia, agar pelanggan dapat melihat dan memesan makanan saya. | Mitra Merchant   | Must-have     |
| US-05  | Sebagai mitra merchant, saya ingin mengedit atau menonaktifkan menu surplus yang sudah habis, agar informasi yang tampil selalu akurat. | Mitra Merchant   | Must-have     |
| US-06  | Sebagai pengguna, saya ingin melihat daftar menu surplus yang tersedia, agar saya bisa memilih makanan yang ingin dipesan.         | User             | Must-have     |
| US-07  | Sebagai pengguna, saya ingin melihat lokasi merchant di peta, agar saya dapat mengetahui merchant terdekat dari posisi saya.       | User             | Should-have   |
| US-08  | Sebagai pengguna, saya ingin mendapatkan petunjuk arah menuju merchant, agar saya bisa sampai ke lokasi pengambilan dengan mudah.  | User             | Could-have    |
| US-09  | Sebagai pengguna, saya ingin memesan makanan surplus dari merchant pilihan saya, agar saya bisa mendapatkan makanan sebelum kehabisan. | User             | Must-have     |
| US-10  | Sebagai mitra merchant, saya ingin mengkonfirmasi atau menolak pesanan yang masuk, agar saya bisa memastikan ketersediaan makanan. | Mitra Merchant   | Must-have     |
| US-11  | Sebagai mitra merchant, saya ingin menandai pesanan sebagai selesai setelah diambil pelanggan, agar riwayat transaksi tercatat dengan akurat. | Mitra Merchant   | Should-have   |
| US-12  | Sebagai mitra merchant baru, saya ingin mengajukan verifikasi akun dengan mengunggah dokumen usaha, agar saya dapat mulai berjualan di EcoEats. | Mitra Merchant   | Must-have     |
| US-13  | Sebagai admin, saya ingin meninjau dan memverifikasi atau menolak pengajuan akun merchant baru, agar hanya merchant sah yang dapat berjualan di platform. | Admin            | Must-have     |
| US-14  | Sebagai pengguna, saya ingin memberikan rating dan ulasan kepada merchant setelah pesanan selesai, agar pengguna lain dapat memilih merchant terpercaya. | User             | Won't-have    |
